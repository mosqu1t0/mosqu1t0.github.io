Python 函数&方法笔记

# Python 函数&方法笔记

提醒：有时候pycharm会自动补齐空格，然后导致报错，记得检测
创建空列表时 a = [] ,创建空字典 a = {}

# 函数

# list()

list()函数是Python的内置函数。它可以将任何可迭代数据转换为列表类型，并返回转换后的列表。当参数为空时，list函数可以创建一个空列表。

# im2col

im2col是一个函数，将输入数据展开以适合滤波器（权重）。

# 方法

# np.argmax()

返回最大值的索引。

# np.size()

主要是用来统计矩阵元素个数，或矩阵某一维上的元素个数的函数。

# np.shape()

返回矩阵的形状。

# np.reshape()

函数用于在不更改数据的情况下为数组赋予新形状。

# list.reverse()

该方法没有返回值，但是会对列表的元素进行反向排序。

# 字典.values()

values() 方法返回 view 对象。这个视图对象包含列表形式的字典值。

该视图对象会反映对字典的任何改变。

# 字典.items()

返回一个字典的文字及值的列表。

# numpy 笔记

# array数组

  • 矩阵

    第一层[]表示一行（一维），即 [7,2,9，10] 表示一个行向量
*特殊*：一层[]默认会被numpy识别为数组，而不是矩阵因此shape为[4,]（这是我为了方便理解的臆想，不准确），但是同样能当作行向量使用。当把shape为 3x1 的列向量转置，会得到shape为 1x3 的矩阵，即行向量。若要把数组（行向量），转置为列向量，需要新增加一个维，即使用 a=a[:np.newaxis] 或者用 a=a.reshape(1,a.size)

第二层[]表示矩阵（两维），即 [[5.2,3.0,4.5],[9.1,0.1,0.3]] 表示2x3的矩阵
*特殊*：[[1],[2],[3]]	表示3x1的矩阵，即列向量

第三层[]表示三维，即[[[1,4,7],[2,9,7],[1,3,0],[9,6,9]],[[2,3,4],[3,4,5],[0,1,2],[7,6,8]]] 表示4x3x2矩阵

如此类推......

    axis表示矩阵的维序数及其方向，0从上往下，1从左往右，2从前往后
ndim是矩阵的维数

  • 广播

    简单总结原则：他们是平等的，或者其中一个是1
(起码一个维元素数量相等，另一个维元素数量为1）
