# bash语言是一门弱类型语言

# 变量

# 变量命名规则

  1. 开头为字母或下划线，名字中间中能有字母、数字和下划线组成;
  2. 变量的长度不超过255个字符;
  3. 变量名在有效的范围内必须是唯一的; //如再次定义则会替换上一个变量的值
  4. 在Bash中,变量的默认类型都是字符串型. //即系统把所有值都当作字符串放到变量中，不论这个“字符串”实际上是整数、浮点数等等
  5. 等号两边不能有空格
  6. 如果值中有空格要用 "" 或 '' 扩起
    • 单引号 扩起的内容完全变成字符， 双引号保留特殊意义

变量默认只会声明字符串类型，使用declare命令才能声明其他类型

# 变量的分类

  • 用户自定义变量

  • 环境变量：保存操作系统中环境相关的数据，可自定义，但对系统生效的环境变量名和变量作用是固定的

  • 位置参数变量：主要用来向脚本当中传递参数或数据，变量名不能自定义，变量作用固定 //实际是预定义变量的一种

  • 预定义变量：bash 中已经定义好的变量，变量名不能自定义，变量作用也是固定的

# 变量查询 和 删除

命令set

  • 直接输入，会显示包括用户定义在内所有的变量

  • -u 设置此选项，调用未声明变量时会报错(默认无任何提示)

命令unset

  • unset直接跟变量名，删除该变量

# 环境变量

# 与用户自定义的区别：

  • 环境变量是全局变量，用户自定义变量是局部变量
  • 用户自定义变量只能在当前的shell中生效，环境变量在当前shell和这个shell的所有子shell中生效

# 用户自定义环境变量

  • export 变量名=变量值
  • export 变量名

# 查询环境变量

命令env

# 常用的环境变量

HOSTNAME:主机名
SHELL:当前的shell
TERN:终端环境
HISTSIZE:历史命令条数
SSH_CLIENT:当前操作环境是用ssh连接的，这里记录客户端ip
SSH_TTY:ssh连接的终端时pts/1
USER:当前登陆的用户
PATH:系统查找命令的路径

**PATH **添加路径

  • PATH="$PATH":/xx/ 临时方法
  • 在/etc/profile 中添加 export PATH="$PATH:xxxxxx"

**PS1 **操作系统的提示符

  • \d：显示日期，格式为”星期 月 日“
  • \H：显示完整的主机名。如默认主机名"localhost.localdomain"
  • \t：显示24小时制时间，“HH:MM:SS”
  • \A：显示24小时制时间，"HH:MM"
  • \u：显示当前用户名
  • \w：显示当前所在目录的完整名称
  • \W：显示当前所在目录的最后一个目录
  • \$：提示符，root为#，普通用户为$

PS2

换行提示符

使用 pstree 可以查看进程树

  • LANG 定义系统主语系的变量

  • LC_ALL 定义整体语系的变量

# 位置参数变量

向脚本文件传递参数

位置参数变量 作用
$n n为数字，$0代表命令本身，$1-9代表第一到第九个参数，十以上要用大括号表示，如9代表第一到第九个参数，十以上要用大括号表示，如9代表第一到第九个参数，十以上要用大括号表示，如{10}
$* 表示命令行中的所有参数，$*把所有的参数看成一个整体
$@ 表示命令行中的所有参数，不过它把每个参数区别对待
$# 表示命令行中所有参数的个数

# 预定义变量

预定义变量 作用
$? 最后一次执行的命令的返回状态。如果这个变量的值为0,证明上一个命令正确执行：如果这个变量的值为非0(具体由命令决定)，则证明上一个命令执行不正确( && || 的原理)
$$ 当前进程的进程号(PID)
$! 后台运行的最后一个进程的进程号(PID)

# 接受键盘输入

向脚本输入

  • read
    • -p "提示信息"：在等待read 输入时，输出提示信息
    • -t 秒数：read命令会一直等待用户输入。使用该选项可以指定等待时间
    • -s ：隐藏输入的数据，适用于机密信息的输入
    • -n：接收的字符数量限制

# 运算

# declare 声明变量类型

  • declare [+/-][选项] 变量名

  • -：给变量设定类型属性

  • +：取消变量的类型属性

    选项

    • a：数组
    • i：整形
    • x：环境变量(export 命令的原理)
    • r：只读变量(无法更改取消删除，但是临时的，避免定义成环境变量或写进系统文件)
    • p：显示指定变量的被声明类型
#声明数组
a[0]=zp
a[1]=tp
declare -a a[2]=live
#访问数组
echo ${a}
echo ${a[2]}
echo ${a[*]}

# 数值运算

#方法1
a=1
b=2
declare -i c=$a+$b

#方法2
#expr 或 let 数值运算工具

c=$(expr $a + $b) #加号两旁必须有空格
let c=$a+$b		#带赋值的运算

#方法3
#$((运算式)) 或 $[运算式]
c=$(($a+$b))
c=$[$a+$b]

a=$(ls) #把命令先执行，结果赋给变量

b=ls #把字符串ls赋予给变量

# 变量测试

非常少用，脚本优化时可以使用，会一个都可以了

变量置换方式 变量y没有设置 变量y为空 变量y设置值
x=${y-新值} x=新值 x为空 x=$y
x=${y:-新值} x=新值 x=新值 x=$y
x=${y+新值} x为空 x=新值 x=新值
x=${y:+新值} x为空 x为空 x=新值
x=${y=新值} x=新值, y=新值 x为空, y不变 x=$y， y不变
x=${y:=新值} x=新值, y=新值 x=新值, y=新值 x=$y， y不变
x=${y?新值} 新值输出到标准错误输出(屏幕) x为空 x=$y
x=${y:?新值} 新值输出到标准错误输出 新值输出到标准错误输出 x=$y
